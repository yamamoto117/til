## GitHub Actionsとは
* あらかじめ定義した処理と条件の組み合わせ(=ワークフロー)を自動化するGitHub公式の機能
* GitHub Actionsではリポジトリに対するプッシュなどの処理をトリガーとして、専用のWorkflowに定義しておいた処理を自動で実行する
* GitHub Actionsを使うことにより、開発の現場で注目されている「CI/CD」の実現も可能

## メリット
* GitHubの仮想サーバーを利用できる
* GitHubに関わる処理の自動化がスムーズに実現できる
