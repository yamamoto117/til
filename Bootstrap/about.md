## Bootstrapとは
* ウェブサイトやwebアプリケーションを作成できるフリーソフト

## Boostrapの種類
* Angular directives for Bootstrap
* Twitter Bootstrap
* Bootstrap Themes
* BootstrapWP

## メリット
* レイアウトのデザインを柔軟に調整できる
* 表示が分かりやすい
* モバイルに対応している

## デメリット
* カスタマイズ知識がある人材が必要
* 機能を詰め込んだ開発には向かない
* 日本語文章の表示の調整が必要
